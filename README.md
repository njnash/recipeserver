### How do I get set up? ###

* Install node.js
* Install npm
* Install packages using npm
    * npm install express
    * npm install serve-favicon
    * npm install morgan
    * npm install cookie-parser
    * npm install body-parser
    * npm install debug
    * npm install jade
    * npm install file-stream-rotator
    * npm install nodemailer
